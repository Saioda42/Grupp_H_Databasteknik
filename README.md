Library Management System

A console-based Library Management System built with Python, PostgreSQL, and SQLAlchemy ORM.
This project was created as part of the Databasteknik PIA25 course.

Features

Book Management
	•	Add new books
	•	Search books by title
	•	View all books
	•	View available books

Member Management
	•	Add new members
	•	Search members
	•	View all members

Loan Management
	•	Register new loans
	•	Register returns
	•	View active loans
	•	View overdue loans
